# Frontend Angular 18

Aplicación Angular 18.

## Estructura

- `src/app/` - componentes y lógica de la aplicación
- `src/app/services/` - servicios HTTP
- `src/environments/` - configuración de entorno

## Ejecución

```bash
npm install
npm run ng -- serve
```

## Build

```bash
npm run ng -- build
```
